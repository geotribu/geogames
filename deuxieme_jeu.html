OPENLAYERS API JavaScript très populaire permettant l'affichage de données géographiques côté client
CACHE Expression qui désigne le fait de préparer à l’avance toutes les images qui peuvent être consultées par un utilisateur sur un portail cartographique, dans le seul but est d'accélérer la consultation
GEOMAJAS Environnement extensible de webmapping qui intègre de façon transparente des algorithmes puissants du côté serveur dans le navigateur web. Il fournit une plateforme pour l’intégration du côté serveur de données géospatiales permettant à de multiples utilisateurs de contrôler et de gérer les données depuis leurs propres navigateurs
PROJ Bibliothèque de fonctions permettant d'effectuer des conversions entre différentes projections cartographiques
MAPFISH Framework flexible et complet pour construire des applications de cartographie en ligne. Il met l’accent sur la haute productivité et un développement de grande qualité. Il fournit des outils spécifiques pour créer des services web qui permettent d’interroger et d’éditer des objets géographiques
TinyOWS Serveur cartographique qui gère le protocole WFS-T, aujourd'hui intégré au projet Mapserver-Suite
GEOS Bibliothèque C++ qui vise à fournir toutes les fonctionnalités JTS, Incluant toutes les spécifications OGC, les fonctions spatiales sous-jacentes et les opérateurs spatiaux, ainsi que des fonctions spécifiques de topologie améliorées
KARTOGRAPH API cartographique qui peut être utilisée soit via une bibliothèque python ou soit via une bibliothèque javascript. L'utilisation du format SVG est le point fort de cette API
MAPNIK Logiciel libre de rendu cartographique, utilisé notamment pour le rendu OSM
ZOOPROJECT Framework pour le Web Processing Service (WPS) adapté pour les développeurs afin de créer et chaîner plusieurs Web Processing Services. Il supporte de nombreux langages de programmation
GEOWEBCACHE Application Java utilisée pour mettre en cache les tuiles de carte provenant d'une variété de sources telles que les WMS
LIZMAP Plugin Qgis qui permet de synchroniser facilement via FTP tous les fichiers nécessaires à l'affichage d'une carte avec QGISserver
POLYMAPS Bibliothèque open source JavaScript permettant de créer des cartes vectorisées dynamiques et interactives au format SVG
MAPFILE Fichier de configuration indispensable pour afficher des données avec MapServer
SLD Schéma XML indiqué par l’OGC pour décrire le style d'une couche
GOOGLEMAPS API cartographique côté client, qui est l'une des plus utilisées au monde.
WMS Service qui sert à retourner une carte visualisable aux formats image tel que le JPEG, le PNG ou le GIF... depuis le serveur.
